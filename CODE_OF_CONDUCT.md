# Code of Conduct

## Our Commitment

TelsonBase exists so that autonomous AI agents earn their autonomy — through demonstrated behavior, human authorization, and a trust model that keeps humans in control. We extend that same respect to our community.

## Standards

### Expected Behavior

- Constructive technical feedback
- Respect for differing viewpoints
- Focus on what's best for the project
- Professional communication

### Unacceptable Behavior

- Harassment or personal attacks
- Trolling or deliberate disruption
- Publishing others' private information
- Conduct inappropriate in a professional setting

## Enforcement

Violations may result in:
1. Warning
2. Temporary ban
3. Permanent ban

Report issues to: support@clawcoat.com

## Scope

Applies to all project spaces: GitHub, Discord, email, events.
